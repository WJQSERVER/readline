# PureGo Readline 路线图

本项目旨在提供一个轻量级、跨平台且功能丰富的纯 Go Readline 库。以下是未来的发展规划：

## v0.1.0 (当前版本 - 已完成)
- [x] 跨平台终端抽象 (Unix/Windows)
- [x] Unicode 支持 (基于 `go-runewidth`)
- [x] 基础行编辑功能 (光标移动、删除)
- [x] 内存历史记录管理
- [x] 基础前缀自动补全
- [x] 基础 ANSI 渲染

## v0.2.0 (短期计划)
- [ ] **持久化历史记录**: 支持将历史记录保存到文件。
- [ ] **更强的补全逻辑**: 支持多列补全展示、模糊匹配。
- [ ] **组合键支持**: 解析更多复杂的 Alt/Meta 快捷键。
- [ ] **搜索功能**: 实现 Ctrl+R 逆向增量搜索。

## v0.3.0 (中期计划)
- [ ] **语法高亮**: 提供钩子函数，支持对输入行进行实时高亮渲染。
- [ ] **多行编辑**: 支持处理换行符，实现复杂的多行输入。
- [ ] **Vi/Emacs 模式**: 提供更强大的编辑模式支持。

## v1.0.0 (长期目标)
- [ ] 性能极致优化，确保在大规模补全时的响应速度。
- [ ] 完善的测试覆盖率。
- [ ] 稳定且易用的插件化 API。
